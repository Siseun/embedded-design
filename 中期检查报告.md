# 三、硬件调试情况

## OLED 显示模块

### 已完成

- OLED 初始化成功
- I2C 通信正常
- 时间显示正常
- 菜单界面显示正常

### 遇到的问题

- 页面刷新速度较高时存在轻微闪烁。

### 解决措施

- 优化页面刷新策略，仅刷新变化区域。

---

## MPU6050 模块

### 已完成

- 成功读取加速度数据
- 成功读取角速度数据
- I2C 通信稳定

### 遇到的问题

- 原始数据波动较大。

### 解决措施

- 增加滑动平均滤波，后续考虑加入卡尔曼滤波算法。

# 四、软件调试情况

## FreeRTOS 任务划分

目前系统划分为以下几个任务：

| 任务 | 功能 |
|------|------|
| taskTimeKeep | 时间维护 |
| taskSensor | MPU6050 数据采集 |
| taskDisplay | 渲染页面 |
| taskMenu | 按键扫描 |
| taskBluetooth | 蓝牙模块 |



目前任务运行稳定，能够正常进行任务切换，无明显死锁或阻塞现象。
