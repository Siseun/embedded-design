### 已完成

- 成功读取加速度数据

# 四、软件调试情况

## FreeRTOS 任务划分

目前系统划分为以下几个任务：

| 任务 | 功能 |
|------|------|
| UI_Task | OLED 页面刷新 |
| Sensor_Task | MPU6050 数据采集 |
| Input_Task | 按键扫描 |
| Time_Task | 时间维护 |

目前任务运行稳定，能够正常进行任务切换，无明显死锁或阻塞现象。
